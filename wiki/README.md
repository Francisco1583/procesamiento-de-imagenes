# Wiki del Proyecto

Esta carpeta contiene la documentación de la wiki para el proyecto de procesamiento paralelo de imágenes BMP.

## Qué cubre

- Interfaz en Python con PyQt5.
- Backend en C/OpenMP.
- Transformaciones, resultados y conclusiones del experimento.
- Guía de compilación y ejecución.

## Estructura

```
wiki/
├── _Sidebar.md
├── _Footer.md
├── Home.md
├── 1-Objetivos.md
├── 2-Contexto-Experimental.md
├── 3-Resultados-y-Datos.md
├── 4-Análisis-Detallado.md
├── 5-Conclusiones.md
├── 6-Arquitectura.md
└── 7-Guía-Instalación.md
```

## Uso

- Abre `Home.md` como punto de entrada.
- Si la vas a publicar en GitHub Wiki, copia estos archivos al repositorio `.wiki`.

## Nota

La wiki fue actualizada para reflejar la migración de la interfaz a Python y las optimizaciones del backend en C.
